가이드

설치, 실행, 기본 사용 흐름을 정리합니다.

1. 다운로드

다운로드 페이지에서 최신 ChatChosim 버전을 받습니다.

2. 실행

압축 해제 후 실행 파일을 실행합니다. 처음 실행 시 보안 안내가 표시될 수 있습니다.

3. 사용 시작

질문, 정리 요청, 초안 작성 요청 등 간단한 작업부터 시작합니다.

SmartScreen 경고가 표시될 수 있습니다. 게시자 인지도에 따라 새 실행 파일에서 흔히 보일 수 있습니다.
