직접 보내주세요.

QueryBot에 추가되었으면 하는 기능, 불편했던 점, 개선 아이디어를 보내면 관리자가 확인 후 검토한다. 구체적으로 적을수록 더 빠르고 정확하게 반영하기 쉽다.

1
구체적으로 적기 어떤 기능이 필요한지, 언제 필요한지, 어떤 식으로 동작하면 좋은지 적는 편이 좋다.
2
문제 상황 설명 버그나 불편 사항이면 재현 상황이나 사용 환경을 함께 적으면 확인이 빨라진다.
3
관리자 검토 보낸 제안은 저장되고 관리자에게 알림이 전송된다. 필요한 경우 피드백이 갈 수 있다.

제안 보내기

아래 항목을 입력한 뒤 보내기 버튼을 누르면 된다.

예: 어떤 상황에서 필요한지, 어떤 방식으로 동작하면 좋은지
보낸 제안은 저장되고 관리자에게 알림이 전송된다.
허위 제보나 테스트 목적의 반복 전송은 처리 대상에서 제외될 수 있다.
