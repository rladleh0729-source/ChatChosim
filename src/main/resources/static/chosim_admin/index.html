데이터 관리

지금은 프론트 기본 화면만 만든 상태다. 나중에 이 버튼을 /api/admin/knowledge 와 연결하면 실제 저장이 된다.

제안 관리

예시 제안 1
질문: 노아벨이 뭐야? / 상태: 대기 중
예시 제안 2
질문: ChatChosim은 무엇인가? / 상태: 대기 중
나중에 사용자 제안은 /api/suggest 로 들어오고, 여기서는 /api/admin/suggestions 로 관리하면 된다.

설정

지금은 화면만 있는 상태다. 이후 /api/admin/settings 와 연결하면 실제 설정 저장이 가능하다.

최근 로그

예시 로그 1
사용자 메시지와 응답 로그가 여기에 표시될 예정이다.
예시 로그 2
나중에 /api/admin/logs 와 연결하면 실제 데이터가 보인다.
예시 로그 3
날짜, 질문, 응답, 상태값 등을 볼 수 있게 확장하면 된다.
